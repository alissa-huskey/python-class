Screencasts¶

You can record a screencast of your terminal activity with asciinema or the windows port PowerSession.

Install¶

Install the asciinema module using your preferred installer.

Listing 472 command line¶
pip install asciinema
Listing 473 command line¶
poetry add asciinema

First install scoop if you haven’t already:

Listing 474 PowerShell¶
iwr -useb get.scoop.sh | iex

Then use scoop to install PowerSession:

Listing 475 PowerShell¶
scoop install PowerSession

Usage¶

Record¶

To start recording run asciinema rec followed by a filename to record to. To finish hit Ctrl-D or type exit.

Listing 476 command line¶
asciinema rec sess.cast

To start recording run PowerShell rec followed by a filename to record to. (You may need to use the command PowerShell.exe instead.) To finish hit Ctrl-D or type exit.

Listing 477 PowerShell¶
PowerSession rec sess.cast

Play¶

You can play back a previously recorded screencast with the command asciinema play followed by the filename to play.

Listing 478 command line¶
asciinema play sess.cast

You can play back a previously recorded screencast with the command PowerShell play followed by the filename to play. (You may need to use the command PowerShell.exe instead.)

Listing 479 PowerShell¶
PowerSession play sess.cast
