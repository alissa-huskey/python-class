Suggested Settings#

Below are some of the settings that I prefer and recommend. Each section includes a description of what the change will do followed by the setting(s) to change.

Instructions#

You can change your settings by either modifying your settings.json file or using the Settings UI, whichever you find easiest.

On the left side of each section is a JSON dictionary containing the relevant settings for the settings.json file.

Add the contents of the dictionary for the setting you want to change from this document into the dictionary in your settings.json file.

  1. Use the Preferences: Open Settings (JSON) command from the command palette to open your settings.json file.

  2. Make sure there’s a , at the end of the last line inside the dictionary in the settings.json file.

  3. Copy the the part of the JSON from this document that is inside the { }.

  4. Paste it inside the dictionary in the settings.json file, just above the last } in the file.

On the right side of each section is a list of relevant settings from the Settings UI.

Each shows the category that the setting is listed under from left of the Settings UI followed by the setting name and value.

  1. Use the Preferences: Open Settings (UI) command from the command palette.

  2. Copy the setting title from this document and paste it in the Search settings input box.

    You can confirm that it is the correct setting by checking the categories on the left of the Settings UI against the path listed to the setting in this document.

  3. Change the value of the setting to match value show on the right of the setting in this document.

Hide Open Editors#

In the Explorer don’t show the “open editors” section.

Settings (UI)

Features ‣ Explorer › Open Editors: Visible

0
Listing 712 settings.json#
{
    "explorer.openEditors.visible": 0
}

Minimap#

Make the minimap prettier and hide it by default. (You can open it using the View: Toggle Minimap command.)

Settings (UI)

Text Editor ‣ Minimap ‣

   Editor › Minimap: Enabled

   Editor › Minimap: Render Characters

   Editor › Minimap: Max Column

200

   Editor › Minimap: Show Slider

always
Listing 713 settings.json#
{
    "editor.minimap.enabled": false,
    "editor.minimap.renderCharacters": false,
    "editor.minimap.maxColumn": 200,
    "editor.minimap.showSlider": "always"
}

Reduce preview mode#

When opening files via Quick Open (⌘P / ⌃P) don’t start in preview mode.

Settings (UI)

Workbench ‣ Editor Management ‣ Workbench › Editor: Enable Preview From Quick Open

Listing 714 settings.json#
{
    "workbench.editor.enablePreviewFromQuickOpen": false
}

Autosave#

Enable auto-save.

Settings (UI)

Text Editor ‣ Files ‣ Files: Auto Save

afterDelay
Listing 715 settings.json#
{
    "files.autoSave": "afterDelay"
}

Disable notices#

Disable tips and startup pages.

Settings (UI)

Workbench ‣ Workbench: Startup Editor

none

Workbench ‣ Appearance ‣ Workbench › Tips: Enabled

Extensions ‣ Python ‣ Python: Show Start Page

Listing 716 settings.json#
{
    "workbench.startupEditor": "none",
    "workbench.tips.enabled": false,
    "python.showStartPage": false
}

Live Share#

Don’t add Live Share guests names to your git commit messages.

Settings (UI)

Extensions ‣ Visual Studio Live Share ‣ Liveshare: Populate Git Co Authors

never
Listing 717 settings.json#
{
    "liveshare.populateGitCoAuthors": "never"
}

Highlight cursor line#

In the editor highlight the line your cursor is currently on.

Settings (UI)

Text Editor ‣

   Editor: Render Line Highlight

all

   Editor: Render Line Highlight Only When Focus

Listing 718 settings.json#
{
    "editor.renderLineHighlight": "all",
    "editor.renderLineHighlightOnlyWhenFocus": true
}

Disable suggestions#

Suggestions are a great feature but they may interfere with retaining information as you’re learning. They can make the interface a bit noisy. These settings disable them, though they can still be triggered using a shortcut key.

Settings (UI)

Text Editor ‣ Editor › Parameter Hints: Enabled

Text Editor ‣ Suggestions

   Editor: Suggest On Trigger Characters

   Editor: Quick Suggestions

1

   Editor: Accept Suggestion On Commit Character

Listing 719 settings.json#
{
    "editor.acceptSuggestionOnCommitCharacter": false,
    "editor.suggestOnTriggerCharacters": false,
    "editor.parameterHints.enabled": false,
    "editor.quickSuggestions": {
        "other": false,
        "comments": false,
        "strings": false
    },
}

Git#

Automatically fetch updates from the default remote of the current Git repository.

Settings (UI)

Extensions ‣ Git ‣ Git: Autofetch

true
Listing 720 settings.json#
{
    "git.autofetch": true,
}
