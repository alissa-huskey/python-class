Hand of Cards#

Exercise 120 (Hand of Cards)

  1. Make a list of cards like:
    cards = ["7H", "QC", "2S", "AD", "3C"]

  2. Iterate over the list and print each card symbol with a space after it.
    Hint: To print without adding a newline use print(... end="")

OUTPUT

    Your hand: 7H QC 2S AD 3C

19

24        for i, value in enumerate(row):
25
26            # handle weird values
27            # None should be an empty string (not "None")
28            # and convert everything else to a string
29            if value is None:
30                value = ""
31            else:
32                value = str(value)
33
34            # the length of the value
35            size = len(value)
36
37            # if its longer than the current longest
38            # then replace it
39            if size > lengths[i]:
40                lengths[i] = size
41
42    # return the list of lengths
43    return lengths
