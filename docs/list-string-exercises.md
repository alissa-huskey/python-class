leet converter
--------------
1. Start with a string like `"hello"`
2. Convert it into a list of individual characters
3. Replace the following characters:
```
    - a     : @
    - s     : $
    - l     : 1
    - i     : !
    - e     : 3
    - o     : 0
    - t     : 7
    - space : _
```
4. Join it back to a string and print the result.

**OUTPUT**
`h3110`
